# Отчёт по автоматизации тестирования

## Запланированные задачи

* разработка UI тестов
* автоматизация формы оплаты
* автоматизация формы покупки в кредит
* интеграция отчётов тестирования
* проверка записей в базе данных

## Реализованные задачи

* реализованы UI тесты для формы оплаты
* реализованы UI тесты для формы покупки в кредит
* реализованы тесты проверки базы данных
* интегрированы отчёты Gradle
* интегрированы отчёты Allure

## Нереализованные задачи

Все запланированные задачи были реализованы.

## Риски

В процессе разработки возникли следующие риски:

* нестабильность браузера
* необходимость настройки Docker и базы данных
* отсутствие тестовой документации

## Итог по времени

Расхождение связано с настройкой окружения и отладкой тестов.
