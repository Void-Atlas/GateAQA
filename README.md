# Дипломный проект по профессии «Тестировщик»

## Описание проекта

Проект содержит автотесты для веб-сервиса покупки тура.

Приложение предоставляет два варианта оплаты:

* **Купить**
* **Купить в кредит**

В рамках проекта реализованы:

* UI-тесты
* проверка записей в базе данных
* генерация отчётов тестирования

---

# Используемые технологии

* Java 11
* Gradle
* JUnit 5
* Selenide
* Allure
* Docker
* MySQL
* JDBC

# Необходимое ПО

Перед запуском проекта необходимо установить:

* **Java 11**
* **Docker**
* **Docker Compose**
* **Git**
* **Allure**

---

# Клонирование репозитория

Склонировать проект:

```
git clone https://github.com/Void-Atlas/GateAQA?tab=readme-ov-file
```

Перейти в директорию проекта:

```
cd GateAQA
```

---

# Запуск инфраструктуры

Запустить контейнеры Docker:

```
docker-compose up
```

Будут запущены:

* MySQL
* эмулятор банковских операций "gate-simulator"
* backend приложения

---

# Запуск приложения

Запустить приложение командой:

```
java -jar aqa-shop.jar
```

Приложение будет доступно по адресу:

```
http://localhost:8080
```

---

# Запуск автотестов

Запуск тестов:

```
./gradlew clean test
```

Для **Windows PowerShell**:

```
.\gradlew clean test
```

После выполнения тестов будет сформирован **Gradle отчёт**:

```
build/reports/tests/test/index.html
```

---

# Генерация Allure отчёта

Для генерации отчёта Allure выполнить команду:

```
.\gradlew allureServe
```

После выполнения откроется **Allure Report** в браузере.

---

# Проверка базы данных

Автотесты выполняют SQL-запросы для проверки записей в таблицах базы данных:

* `payment_entity`
* `credit_request_entity`
* `order_entity`

Проверяются статусы операций:

* `APPROVED`
* `DECLINED`

---

# Очистка базы данных

После выполнения каждого теста база данных очищается автоматически методом:

```
SQLHelper.cleanDatabase()
```

---

# Остановка контейнеров

После завершения работы контейнеры можно остановить командой:

```
docker-compose down
```
